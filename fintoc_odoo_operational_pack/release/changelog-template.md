# Changelog

## 17.0.1.0.0 - Initial release
- Added Fintoc payment provider integration for Odoo 17.
- Added redirect checkout flow with checkout session creation.
- Added webhook validation with HMAC SHA256 and timestamp tolerance.
- Added idempotent webhook event processing model.
- Added refund creation and refund cancellation request support.
- Added merchant UX warnings for missing credentials/security requirements.
